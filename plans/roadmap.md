tracked as their own issues:
- ~~**Calendar read-only web view**~~ **DONE (#195)** ‚Äî built as **per-person read-only ICS feeds**, not a custom web page (the tool that works across Apple **and** Google/Yandex, which only subscribe by ICS URL).
